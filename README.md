# Book Catalog Web Project

## Overview
A dynamic web application to manage and browse a catalog of books. Users can register, add, display, search, and remove books through a PHP backend with an SQL database.

## Features
- Backend database functionality for content management
- User registration and login
- Client-side form validation using JavaScript
- Search and filter books by title, author, or genre
- Responsive design using HTML and CSS
- Dynamic behavior based on user input

## Project Structure
- `index.php` – Main page
- `pages/` – HTML pages
- `scripts/` – JavaScript files for form validation and dynamic behavior
- `server/` – PHP backend files
- `database/` – SQL scripts for creating and populating the database
- `styles/` – External CSS files
- `images/` – Images used in the project

## How to Run
1. Install XAMPP or WAMP and start the Apache and MySQL servers.
2. Import the SQL file in `database/book-catalog.sql` into your database.
3. Place the project folder inside `htdocs` (for XAMPP) or the web root.
4. Open `index.php` in your browser to start using the application.

## Technologies Used
- HTML5, CSS3
- JavaScript (DOM Manipulation)
- PHP
- MySQL / SQL
- XAMPP (or similar local server)
